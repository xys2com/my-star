### 一些内容

#### 1、使用 canvas 动画较多、css 动画较少

#### 2、canvas 动画离屏绘制，节俭性能

#### 3、动画执行器 requestAnimationFrame 所执行的内容集中控制。在一个组件内部调用，其他组件将 方法 和 参数 提交到父组件，由父组件统一执行。
